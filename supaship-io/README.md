
# Startup project

In the supabase-io-course folder,

`yarn add vite`
`yarn create vite`

- framework: react
- select name: supaship-io
- variant: typescript

Then use `yarn` to install dependencies.

# First run

`cd supaship-io`
`yarn dev`

# Supabase dependencies

To add supabase dependencies to the project:

`yarn add -D supabase`

`supabase init`

`supabase start`
